FinTrack is a simple command-line application built with Python and SQLAlchemy that helps users manage their expenses, subscriptions, and budgets. Users can easily add, update, delete, and search for expenses, as well as set monthly budgets and receive alerts when they exceed their limits.

## Features

- **Add Categories**: Organize expenses by categories.
- **Add Expenses**: Track individual expenses with details such as title, amount, date, and category.
- **Update Expenses**: Modify existing expense entries.
- **Delete Expenses**: Remove expenses from the record.
- **Search Expenses by Date**: View expenses on a specific date.
- **Add Subscriptions**: Keep track of recurring expenses.
- **Set Monthly Budget**: Define a spending limit for each month.
- **Budget Alert**: Get notified if expenses exceed the set budget for the month.

## Requirements

- Python 3.x
- SQLAlchemy
- SQLite (comes built-in with Python)
